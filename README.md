# 项目运行：
- npm install
- npm run build
- npm run dev

# 项目结构：
src
- common 共公类（数据、环境、公共方法）
- component 项目组件
- images 图片地址(目前没用到)
- pages 项目页面
- styles css目录
- vendor 第三方依赖(目前没用到)
- view 模板(目前没用到)
